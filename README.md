# DASHBOARD OPERACIONAL FRONTEND

## PRÉ-REQUISITOS

- Node.js (versão (verificar) ou superior)
- npm

### INSTALAÇÃO

- _Clone o repositório_
  - git clone https://tools.ages.pucrs.br/dashboard-operacional/frontend.git
  - cd frontend

- _Instalação das dependências_
  - npm install

### COMO EXECUTAR

    - npm run dev
