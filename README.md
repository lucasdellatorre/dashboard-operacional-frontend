# DASHBOARD OPERACIONAL FRONTEND

## PRÉ-REQUISITOS

- Node.js (versão (verificar) ou superior)
- npm

### INSTALAÇÃO

- _Clone o repositório_

  - git clone https://github.com/seu-usuario/frontend.git
  - cd frontend

- _Instalação das dependências_
  - npm install

### COMO EXECUTAR

    - npm run dev
